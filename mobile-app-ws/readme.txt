# Spring Boot User CRUD Application

This is a simple Java Spring Boot application that provides CRUD operations for managing user data. The application stores user information temporarily in-memory using a HashMap.

## Prerequisites

Before running the application, ensure you have the following installed:

- Java 8 or later
- Maven

### API Endpoints

- Get User by Id: http://localhost:8080/users/{userId}

- Create User: http://localhost:8080/users

- Update User: http://localhost:8080/users/{userId}

- Delete User: http://localhost:8080/users/{userId}
